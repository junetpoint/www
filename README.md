## www

Hey! This is my personal website, built with [Astro](https://astro.build). Here you'll find a bit about me and some of my projects. I also post articles on the built-in blog from time to time. Feel free to use this site as inspiration or a template for your own ideas!

```shell
# 1. Install the dependencies
pnpm install

# 2. Launch the server
pnpm dev
```
